## Кратко

`console` — это объект, который используется для доступ к методам отладки и логирования.

Методы `Console API` можно разделить на несколько категорий:
- логирование;
- функции подсчёта;
- функции группировки;
- таймеры.

Реализация и поведение методов `Console API` зависит от среды выполнения.
Объект `console` доступен в коде как свойство глобального объекта: в браузере - `window`, в Node.js - `global`.

## Методы логирования

### `assert()`

Логирует сообщение в случае если результат выражения, переданного в качестве первого аргумента - `false`:

```js
function checkNumber(num) {
  const value = +num
  console.assert(!Number.isNaN(value), 'это не число!')
  return value
}

checkNumber('a')
// Assertion failed: это не число!
```

### `clear()`

Очищает консоль от ранее выведенных сообщений.
```js
console.log('Внимание!')

console.clear()
```

### `debug()`





Когда JavaScript код выполняется в браузере, взаимодействие с консолью осуществляется с помощью инструментов разработчика.
